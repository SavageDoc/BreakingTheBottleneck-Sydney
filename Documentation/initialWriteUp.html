A comparison of linear regression and random forests was performed on house price data. The data set includes 506 observations across 14 variables, including both dependent and independent variables.

A number of parameters were defined in support of this problem. A summary of these values is given below.

Parameter Value Comments
Random seed 100 For reproducibility.
Split percent 75 Fraction of the data to use as training
Exclude correlations? TRUE For linear model.
Correlation threshold 0.6 NA if not excluding correlations
Number of trees 200 For random forest
Variables at each split 5 For random forest
Minimal split size 10 For random forest
